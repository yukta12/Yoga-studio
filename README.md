# Yoga-Studio

A Modern HTML5 Template for Yoga Website.

## Features

- Bootstrap 3
- Easy Documentation
- Easy to make changes

## Getting Started
The easiest way to get started is to clone the repository

```bash
git clone https://github.com/yukta12/Yoga-studio.git
# Run the index file
- index.html 
```
